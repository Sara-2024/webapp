## 2026-02-14 10:20

### タイムスタンプ問題の修正

**問題**: チャートの日付が現在時刻より2日先（2月16日）を表示していた

**原因**:
- ローソク足生成時のタイムスタンプが未来の時刻になっていた
- `generateCandle`関数が前のローソク足のタイムスタンプ + 60秒を使用
- 最初のローソク足が誤った時刻で生成された場合、全ての後続ローソク足も誤った時刻になる

**対応**:
1. 全てのローソク足とサインを削除
2. コード修正: 最新ローソク足が現在時刻から10分以上離れている場合、タイムスタンプをリセット
3. 新しいローソク足を正しい現在時刻で再生成（134本）

**修正コード**:
```typescript
// タイムスタンプの検証: 最新ローソク足が現在時刻から大きく離れている場合はリセット
const now = Math.floor(Date.now() / 1000)
let candleToUse = latestCandle

if (latestCandle) {
  const timeDiff = Math.abs(now - latestCandle.timestamp)
  // 10分以上離れている場合は、新規生成として扱う
  if (timeDiff > 600) {
    candleToUse = null
  }
}
```

**結果**: 
- タイムスタンプが正しく修正され、チャート表示が現在時刻と一致
- ローソク足総数: 134本（2026-02-14 10:10～12:23 JST）
- 今後は外部Cronサービスによって自動的に正しい時刻でローソク足が生成される

**影響範囲**: 本番環境（webapp-production D1データベース）
